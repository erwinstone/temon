
# Temon

**T**ypescript + **E**sbuild + Node**mon**

Typescript runner, bundler, watcher powered by esbuild & nodemon


## Installation

Install my-project with npm

```bash
  npm install temon
```
    
## Usage/Examples

```bash
temon index.ts
temon index.ts --build dist/index.js
temon index.ts --watch
```
